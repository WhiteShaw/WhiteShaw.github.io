centos6 搭建博客

Posted on 2019-03-14 | In 技术类

参考连接：https://segmentfault.com/a/1190000012907499

问题：

1. centos6 的版本比较老，安装nodejs按照教程不好使，使用如下命令：

sudo rpm -ivh http://download.fedoraproject.org/pub/epel/6/x86_64/epel-release-6-8.noarch.rpm
sudo rpm –import /etc/pki/rpm-gpg/RPM-GPG-KEY-EPEL-6
sudo yum -y install nodejs npm –enablerepo=epel

2. 如果服务器的端口不是标准的22，则需要进行以下修改：

cat>~/.ssh/config

host newdomain
hostname domain.com
port 3022
